# 🛒 E-commerce Sales Dashboard - Power BI

## 📌 Project Overview
This project presents an interactive **E-commerce Sales Dashboard** built using Power BI.  
The main objective is to analyze sales performance, identify trends, and generate actionable business insights.

The dashboard helps in understanding customer behavior, product performance, and overall business growth.

---

## 🎯 Objective
The objective of this project is to analyze E-commerce sales data and identify key insights such as:
- Sales trends over time  
- Top-performing categories and products  
- Customer purchasing patterns  
- Payment mode preferences  

This helps businesses make data-driven decisions and improve sales performance.

---

## 📊 Key Performance Indicators (KPIs)
- Total Sales Amount  
- Total Quantity Sold  
- Total Profit  
- Average Order Value (AOV)  

---

## 📈 Key Features
- 📍 State-wise Sales Analysis  
- 📦 Category & Sub-category Performance  
- 📅 Monthly Profit Trend Analysis  
- 💳 Payment Mode Distribution (COD, UPI, Credit Card, Debit Card, EMI)  
- 🎯 Interactive Filters (Quarter, State)  
- 📊 Dynamic Visualizations for better insights  

---

## 🛠 Tools & Technologies Used
- Power BI  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Data Visualization Techniques
